## üì• Installation Instructions

### Prerequisites
- Unlocked bootloader
- Backup of your current boot image
- Existing root access (Magisk / KernelSU / APatch)

### Via Kernel Flasher
1. Download the appropriate ZIP for your device from the Assets below
2. Flash the ZIP using **Kernel Flasher** or **Horizon Kernel Flasher**
3. Reboot your device
4. Install **KernelSU-Next Manager** from the link above
5. Install **SUSFS Module** from within the manager
